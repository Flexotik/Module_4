# Трекер прочитанных книг

Консольное приложение для учёта прочитанных книг с модульной архитектурой.

## Функциональность
- Добавление книги (с проверкой дубликатов)
- Просмотр всех книг
- Средняя оценка
- Статистика по авторам
- Удаление книги
- Хранение данных в `books.json`

## Структура модулей
- `main.py` — интерфейс и меню
- `models.py` — класс `Book`
- `storage.py` — работа с JSON
- `stats.py` — статистика и расчёты

## Запуск
```bash
python main.py
